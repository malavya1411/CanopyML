# 🌍 CanopyML

> **Satellite Land Cover Classification & Deforestation Detection using ResNet50 Transfer Learning**

A complete end-to-end research pipeline that trains a CNN on EuroSAT satellite imagery, applies it to
real Sentinel-2 imagery to build land cover maps, detects deforestation between two time periods, and
validates results against Global Forest Watch (GFW) data.

---

## 📋 Pipeline

```
EuroSAT (27k patches, 10 classes)
         │
         ▼
ResNet50 ──► Stage A: Transfer Learning (head only, 10 epochs, lr=1e-3)
         └──► Stage B: Fine-Tuning (all layers, 10 epochs, lr=1e-4 + early stopping)
                           │
              Sentinel-2 2018 + 2024 GeoTIFFs
                           │
              patchify → GPU inference → reconstruct grid
                           │
              Change Detection (Forest → Non-Forest)
                           │
              GFW Validation (Precision / Recall / F1 / IoU)
```

## 🚀 Quick Start (Google Colab)

1. Open `CanopyML_Deforestation_Detection.ipynb` in Google Colab.
2. Set runtime to **T4 GPU** (Runtime → Change runtime type).
3. Upload datasets:
   - EuroSAT RGB → `/content/EuroSAT_RGB/` (standard ImageFolder structure)
   - Sentinel-2 imagery → `/content/sentinel_2018.tif`, `/content/sentinel_2024.tif`
   - GFW loss mask → `/content/gfw_loss.tif`
4. Run all cells top-to-bottom.

> **Note:** Cells include graceful fallbacks with synthetic demo data if satellite files are not yet uploaded, so all pipeline stages can be tested immediately.

## 📊 Model Configuration

| Hyperparameter | Value |
|---|---|
| Backbone | ResNet50 (ImageNet pretrained) |
| Stage A lr | 0.001 |
| Stage A epochs | 10 |
| Stage B lr | 0.0001 |
| Stage B epochs | 10 (+ early stopping, patience=3) |
| Batch size | 32 |
| Optimizer | Adam |
| Loss | CrossEntropyLoss |
| Patch size (inference) | 64×64 px |
